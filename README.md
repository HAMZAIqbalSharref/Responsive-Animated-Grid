# Animation Test 1

This project demonstrates a responsive animated layout using HTML and CSS. It features:

- Scattered falling text animation
- Responsive grid layout for desktop, tablet, and mobile
- Custom styles for each device size

## How to Use

1. Open `index.html` in your browser.
2. View on different devices or resize your browser to see the responsive design.

## Features
- Modern CSS grid and flexbox layout
- Media queries for laptops, tablets, and phones
- Animated text effects

## Customization
- Edit `Style.css` to change colors, fonts, or animations.
- Add your own images to the `image/` folder.
